**効果**:

- 12回呼び出し: **48ノード削減**

### 2. `expected_prod_pos_expr()` の削減

**最適化内容**:
- `phi2_expr()` の最適化により、`expected_prod_pos_expr()` 全体のノード数が削減

**効果**:
- `expected_prod_pos_expr()` 1回あたり: 4ノード削減（79 → 75）
